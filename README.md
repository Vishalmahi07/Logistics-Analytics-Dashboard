# Strategic Logistics Analytics Dashboard 

## Project Overview
This project provides an end-to-end analytics solution for a national logistics network. By integrating shipment, hub, and financial data, the dashboard identifies operational bottlenecks and cost-saving opportunities.

## 📊 Key Business Objectives
* **Improve On-Time Delivery:** Goal of 90% SLA compliance (Current: 81.6%).
* **Cost Reduction:** Analyzing the $1.35M spend to reach a $2.15/KM efficiency target.
* **Hub Optimization:** Identifying capacity constraints in high-volume hubs like Manchester and London.

## 🛠️ Tech Stack
* **Power BI:** Data Modeling, DAX, and Visualization.
* **Excel/CSV:** Source data for Shipments, Hubs, and Costs.
* **GitHub:** Project Documentation and Version Control.

## 💡 Key Insights
1. **The "March Dip":** Identified a performance collapse to 78% due to a 15% volume surge.
2. **Efficiency Gap:** The East Midlands is the least efficient region ($1,844/shipment) compared to the North West ($1,633/shipment).
3. **Hub Stress:** Manchester and London process 40% of volume but account for 60% of SLA breaches.

## Recommendations
* **Load Rebalancing:** Shift 10% volume to "Gold Tier" hubs (Edinburgh/Bristol).
* **Cost Audit:** Immediate carrier renegotiation in the East Midlands.
* **Dynamic Planning:** Implement flex-capacity 2 weeks before seasonal peaks.
